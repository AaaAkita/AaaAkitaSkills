# Code Review Skill（代码审查技能）

<!-- 进行代码审查时，按以下步骤执行 -->
When reviewing code, follow these steps:

## Review checklist（审查清单）

1. **Correctness（正确性）**: Does the code do what it's supposed to? / 代码是否实现了预期功能？
2. **Edge cases（边界情况）**: Are error conditions handled? / 错误条件和边界情况是否已处理？
3. **Style（代码风格）**: Does it follow project conventions? / 是否遵循项目编码规范？
4. **Performance（性能）**: Are there obvious inefficiencies? / 是否存在明显的性能问题？

## How to provide feedback（如何提供反馈）

- Be specific about what needs to change / 明确指出需要修改的内容
- Explain why, not just what / 说明原因，而不仅仅是指出问题
- Suggest alternatives when possible / 尽可能提供替代方案
